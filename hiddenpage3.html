Decoding Decisions with If-Else

In programming, you often need to make decisions. Java provides if-else statements to help your code make choices based on conditions. Let's explore this with an example!


If-Else Example: The Adventure of Code

Letâ€™s see if our hero, CodeMaster, has enough energy to complete their mission!

public class HeroMission {
    public static void main(String[] args) {
        int energyLevel = 75;  // Hero's energy level

        // Check if the hero has enough energy to complete the mission
        if (energyLevel >= 50) {
            System.out.println("CodeMaster is ready for the mission!");
        } else {
            System.out.println("CodeMaster needs to recharge!");
        }
    }
}

Explanation:

  • if checks if energyLevel is 50 or more. If it is, CodeMaster is ready!
  • If energyLevel is less than 50, the else block will tell CodeMaster to recharge.
