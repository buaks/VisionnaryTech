The Array Adventure

In Java, an array allows you to store multiple values in a single variable. Think of it as a collection of treasures. Let's dive into the array adventure with CodeMaster!


Array Example: CodeMaster’s Quest for Gems

Our hero, CodeMaster, collects gems. Let’s create an array to store the names of these magical gems!

public class GemCollector {
    public static void main(String[] args) {
        // Array to store gem names
        String[] gems = {"Ruby", "Emerald", "Sapphire", "Diamond", "Amethyst"};
        
        // Loop through the array and print each gem
        System.out.println("CodeMaster’s Gem Collection:");
        for (int i = 0; i < gems.length; i++) {
            System.out.println(gems[i]);
        }
    }
}

Explanation:

  • An array named gems holds five different types of gems.
  • The for loop iterates through each element in the array and prints out the gems.
  • gems.length is used to get the total number of gems in the array.
