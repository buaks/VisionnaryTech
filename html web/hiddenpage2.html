Mastering Variables and Data Types in Java

Understanding how to store and manipulate data is crucial in any programming language. Letâ€™s dive into the basics of Java variables and data types:


Java Data Types

  • int: Used to declare whole numbers (e.g., int age = 25;)
  • double: For real numbers with decimal points (e.g., double score = 98.6;)
  • String: Stores text data enclosed in double quotes (e.g., String name = "Alice";)
  • char: Holds a single character (e.g., char grade = 'A';)
  • boolean: Represents true/false values (e.g., boolean is Online = true;)

Example:

public class DataTypes {
    public static void main(String[] args) {
        String name = "Archie";
        int age = 18;
        double score = 9.9;
        char group = 'S';
        boolean online = true;
        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
        System.out.println("Score: " + score);
        System.out.println("Group: " + group);
        System.out.println("Online: " + online);
    }
}
